# [Polly](https://pol-ly.netlify.app/)

[![Netlify Status](https://api.netlify.com/api/v1/badges/b5c8758b-a81b-416b-8b6c-295dfb0bac71/deploy-status)](https://app.netlify.com/sites/pol-ly/deploys)

Make your polls and cast your votes

## Tech stack I used

- MongoDB for data storage

- Express and NodeJS for back-end service

- React with Hooks and Context

- Bulma CSS for layout
